This is a quick reference guide for common data aggregation functions often used in data analysis (e.g., with Python's Pandas library or SQL).

## Function List

1. **`sum()`**
   - Calculates the total sum of the values.

2. **`mean()`**
   - Calculates the arithmetic mean (average) of the values.

3. **`count()`**
   - Counts the number of non-null items or observations.

4. **`min()`**
   - Finds and returns the minimum (lowest) value in the dataset.

5. **`max()`**
   - Finds and returns the maximum (highest) value in the dataset.

6. **`std()`**
   - Calculates the standard deviation, which measures the amount of variation or dispersion of a set of values.
